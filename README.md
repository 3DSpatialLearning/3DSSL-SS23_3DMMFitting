## Installation

### Windows/Linux
```bash
pip install -e .
pip install -r requirements.txt
```
Install pytorch3d following the steps from: https://github.com/facebookresearch/pytorch3d/blob/main/INSTALL.md

### MacOS
```bash
pip install -e .
pip install -r requirements-mac.txt
```
## Conda environment setup
